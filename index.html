Projetos em Ciência de dados

Construção de um modelo de Rankeamento de Clientes para venda cruzada de seguro automotivo

O objetivo é vender aos clientes de uma empresa de seguros de saúde a oferta de um novo produto: Seguro de Automóveis. Para atingir essa meta, criei um projeto que se baseia no funcionamento de um modelo de machine learning de classificação, com a particularidade de otimizar a eficácia da equipe de vendas através do rankeamento da base de dados de clientes de acordo com a probabilidade de cada um em responder positivamente quanto a aquisição conjunta da novo produto ofertado (cross selling).

Com o modelo em produção, a expectativa é de que ele seja pelo menos 2.6x mais eficaz em identificar clientes com intenção de compra cruzada de seguros quando em comparação a ausência de modelo (Contato realizado de forma aleatória).

Previsão de Vendas de Lojas utilizando Regressão

Para esse projeto foram utilizados conceitos de programação em Python para manipulação de dados voltados à validação de hipóteses sobre os dados para posterior aplicação em algoritmos supervisionados de regressão em Machine Learning, viabilizando a construção de um modelo que estima o faturamento de cada loja registrada na base de dados de uma empresa do ramo farmacêutico (Rossmann Drogeriemarktt) para as próximas 6 semanas. O objetivo final, é identificar quais lojas teriam faturamento suficiente para realizar reformas em sua estrutura.

O resultado final do Modelo foi um modelo em produção hospedado em nuvem e disponibilizado para consulta em tempo real através de um bot no Telegram. O bot pode ser acessado por qualquer dispositivo conectado a internet com conta no aplicativo, permitindo fácil integração de uso por parte do time de negócios.

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros, e limiares entre overfitting e underfitting de vários algoritmos dentro das tarefas supervisionadas (Classificação e Regressão) e não supervisionadas(agrupamento) através de um ensaio de modelos Machine Learning.
